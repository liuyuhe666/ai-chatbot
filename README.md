## 🧑‍💻 安装和运行

- 安装依赖

```bash
pnpm i
```

- 本地开发

```bash
pnpm dev
```

## 😎 定制

`fork` 该项目，修改 `data/chatbot.json`

```json
[
  {
    "id": 1,
    "name": "",
    "description": "",
    "prompt": ""
  }
]
```
